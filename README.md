## 一.项目概述
一个基于SSM框架构建的maven子父工程。项目用于对城市公交系统的管理，共提供两种登录入口：  
**普通司机登录**：普通司机账号权限相对较小，只能够查看司机的相关信息和修改个人信息，无法对上一级的线路列表和片区模块进行查询和其他操作，更无法查看管理员账号信息。  
**管理员账号登录**：管理员账号拥有着系统的最高权限，可以实现对司机、线路、片区等所有模块的管理，甚至可以设置新的管理员账号。
